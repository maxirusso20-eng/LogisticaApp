# LogisticaApp

App de logística con panel admin + app para choferes.

## Setup para nuevo cliente

1. Crear proyecto en Supabase
2. Copiar `.env.example` → `.env`
3. Completar `.env` con las credenciales del cliente
4. Cambiar `name`, `slug` y `owner` en `app.json`
5. `npm install`
6. `npx expo start`
